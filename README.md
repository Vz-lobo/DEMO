# ☕ Aroma Premium - Landing Page de Café Artesanal

Bienvenido a **Aroma Premium**, una landing page moderna y responsive diseñada para promocionar y vender café artesanal colombiano.

Este proyecto fue desarrollado utilizando **HTML5, CSS3 y JavaScript**, con un enfoque en una interfaz elegante, intuitiva y optimizada para brindar una excelente experiencia de usuario.

---

## 🚀 Características

* Diseño moderno y minimalista.
* Totalmente responsive (PC, tablet y móvil).
* Hero principal con llamada a la acción.
* Sección de productos destacados.
* Beneficios del café artesanal.
* Botones de compra y contacto mediante WhatsApp.
* Iconos de redes sociales.
* Animaciones e interacciones con JavaScript.
* Código limpio, organizado y fácil de modificar.

---

## 🛠 Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Font Awesome

## ▶️ Cómo ejecutar el proyecto

1. Clona este repositorio:

```bash
git clone https://github.com/TU-USUARIO/TU-REPOSITORIO.git
```

2. Entra al directorio del proyecto:

```bash
cd TU-REPOSITORIO
```

3. Abre el archivo `index.html` en tu navegador o utiliza la extensión **Live Server** en Visual Studio Code.
